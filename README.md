# Проект интернет-магазина (учебный)

## Реализованный функционал

### 1. Работа с товарами (класс Product)
- Создание товаров с характеристиками:
  - Название
  - Описание
  - Цена
  - Количество в наличии
- Автоматический подсчет общего количества уникальных товаров
- **Новый функционал:**
  - Сложение товаров одного типа (с проверкой классов)
  - Защита от сложения товаров разных категорий
  - Генерация TypeError при несовместимых операциях

### 2. Управление категориями (класс Category)
- Создание категорий товаров:
  - Название категории
  - Описание категории
  - Список товаров в категории
- Автоматический подсчет:
  - Количества категорий
  - Количества уникальных товаров
  - Общего количества всех товаров в магазине
- **Новый функционал:**
  - Защита от добавления не-товаров в категории
  - Проверка типа добавляемых объектов (только Product и наследники)
  - Генерация TypeError при попытке добавить неподходящий объект
  
### 3. Специализированные классы товаров
- **Smartphone** (наследник Product):
  - Производительность (efficiency)
  - Модель
  - Объем памяти
  - Цвет
- **LawnGrass** (наследник Product):
  - Страна-производитель
  - Срок прорастания
  - Цвет

### 4. Базовый класс продукта (`BaseProduct`)
- **BaseProduct** (Базовый класс для всех продуктов магазина. Реализует):
    - защищённое хранение цены
    - валидацию данных
    - единый интерфейс для продуктов
  
### 5. Миксин для логирования (PrintMixin)
- **PrintMixin** (Миксин для автоматического логирования и строкового представления. Добавляет):
    - логирование создания объектов
    - автоматический repr
    - стандартизированный вывод

### 6. Основной скрипт (main.py)
- Демонстрация работы системы:
  - Создание тестовых товаров
  - Формирование категорий
  - Вывод информации о товарах и категориях
  - Отображение системных счетчиков
  - Примеры:
    - Корректного сложения товаров
    - Обработки ошибок при недопустимых операциях

### 7. Тестирование
- Написаны unit-тесты для проверки:
  - Корректности создания объектов
  - Правильности работы счетчиков
  - Формата выводимой информации
  - Обработки граничных случаев
  - **Новые тесты:**
    - Проверка сложения товаров одного класса
    - Проверка ошибок при сложении разных классов
    - Проверка добавления только продуктов в категории
    - Проверка ошибок при добавлении не-продуктов
