# Проект интернет-магазина (учебный)

## Реализованный функционал

### 1. Работа с товарами (класс Product)
- Создание товаров с характеристиками:
  - Название
  - Описание
  - Цена
  - Количество в наличии
- Автоматический подсчет общего количества уникальных товаров

### 2. Управление категориями (класс Category)
- Создание категорий товаров:
  - Название категории
  - Описание категории
  - Список товаров в категории
- Автоматический подсчет:
  - Количества категорий
  - Количества уникальных товаров
  - Общего количества всех товаров в магазине

### 3. Основной скрипт (main.py)
- Демонстрация работы системы:
  - Создание тестовых товаров
  - Формирование категорий
  - Вывод информации о товарах и категориях
  - Отображение системных счетчиков

### 4. Тестирование
- Написаны unit-тесты для проверки:
  - Корректности создания объектов
  - Правильности работы счетчиков
  - Формата выводимой информации
  - Обработки граничных случаев
